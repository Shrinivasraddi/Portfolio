The Chapri Store

E-commerce Website using MERN Stack

Project Overview

The E-commerce Website Project is a modern online shopping platform developed using the MERN stack (MongoDB, Express, React, Node). It tackles key e-commerce challenges such as SEO, sorting, user experience, and transaction delays. Key features include secure authentication, optimized product filters, fast transactions, logistics integration, and ML-based recommendations. Future goals include AI personalization and enhanced security for better scalability and usability.

My Role: I handled the MongoDB database, ensuring efficient data storage and secure backend operations. I also assisted in frontend development and contributed to the UI/UX design for a smooth and responsive user experience.

Tech Stack

  • MongoDB
  • Express.js
  • React.js
  • Node.js
